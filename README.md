# Digital-Tools-Buying-Website

**A modern digital tools buying marketplace** where users can browse premium SaaS tools, templates, and productivity resources with a sleek shopping cart experience.

Built with clean design, smooth interactions, and fully responsive layout — inspired by professional Figma designs.

## ✨ Features

- **Dynamic Section Toggling** — Easily switch between **Products** and **Cart** using two beautiful central buttons (Product section visible by default).
- **Fully Functional Cart System** — Add products with "Buy Now", view items, remove individual products, see real-time cart count in navbar, and clear cart with "Proceed to Checkout".
- **Modern Product Showcase** — Beautiful 3-column responsive product cards with icons, tags (Popular, New, Best Seller), features list, pricing, and toast notifications powered by **React-Toastify**.
- Used **React-Toastify** for all user feedback (Add to cart, Remove, Checkout success)
- Individual product remove from cart
- Live cart count update in navbar
- "Proceed to Checkout" clears the entire cart with success toast

## 🛠️ Technologies Used

- **React.js**
- **Tailwind CSS + DaisyUI**
- **JavaScript (ES6+)**
- **React-Toastify** (for all add/remove/checkout alerts)
- **JSON** (for mock product data)
